QR Code Generator
*******************

This QR code generator is using Google's Vision API to generate QR codes.
----------------
It generates different type of QR codes, like: 
1. Text
2. URL
3. Email
4. Phone
5. SMS
6. Contact 
7. UPI
8. WIFI
-------------
Explore the project at: [https://qr-avi.herokuapp.com/](https://qr-avi.herokuapp.com/)
